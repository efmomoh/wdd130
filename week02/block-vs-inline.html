Task 2: Nesting Inline in Block

A paragraph <p> element (block) with a <span> element;

Add another inline element like <em> or <strong> inside the paragraph. Observe how it flows within the line. OBSERVATION: The "span tag" element tag inside the "p tag" block element do not start on a new line, it only takes up the space given by the "span element" and aligned inside of the "p tag" block level. Interestingly, the block element takes up the entire width of the screen while the "span element" alings inside of it.

Task3: Nesting Block in Inline

Try placing a
<div> inside a
<span> element.

What error or layout issue do you see? Why does this happen? BECAUSE: a "div" is a block level element and always start on a new line and takes on the entire width of the screen, so when a "div" element is placed or nested inside an inline element-it starts on a new line and takes up full width of the screen.
